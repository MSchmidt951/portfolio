WiFi LED Strip

Description

!Gif of LED strip rainbow
This project uses an ESP8266 to control multiple individually addressable RGB LED strips and create a WiFi network that can be connected to. The user then can change the settings of the LED strips through a web page while connected to the ESP8266.

Skills

  • Use of Internet of Things (IoT) devices
  • Interfacing hardware with microcontrollers
  • Web design
