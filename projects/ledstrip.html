WiFi LED Strip

Description

This project uses an ESP8266 to control multiple individually addressable RGB LED strips and create a WiFi network that can be connected to.
The user then can change the settings of the LED strips through a web page while connected to the ESP8266.

Skills

  • Use of Internet of Things (IoT) devices
  • Web design for interfacing electronics

Closer look

After the LED strips are plugged in to the device and connected to power the user can connect to a WiFi network where they can access a site to control the LED strips.
On this website the user can set the lights of the LED strips, which can be set independently.

Hardware

The LEDs on the LED strip are individually addressable and only require one microcontroller pin to control per LED strip. Two LED strips are supported but many more can easily be added.
The microcontroller used is the ESP8266. This is able to be programmed with Arduino code and can create its own WiFi networks and have websites that can call functions which can change the state of the hardware.
The electronics are mounted on a perfboard which has a 5 volt regulator. The board can be powered by battery through the regulator, or with USB.

Software

The top of the program includes the WiFi name and password, pin numbers and LED strip settings.
In the setup the WiFi network is created and the IP address of the website is set.
Request handlers are used to add functionality to the requests from the website to set the mode of the LED strips.
Three modes have been implemented: solid, rainbowsolid and rainbow. Solid sets the whole LED strip to one colour, rainbowsolid sets the LED strip to a colour then changes the hue, and rainbow creates a wave of colour that travels through the LED strip. All modes can have the delay set to adjust the speed of the colour change.
