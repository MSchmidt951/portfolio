Small RC car

Description

The aim of this project was to design and make a prototype of a remote control car that would be as small as possible.
I achieved this by 3D printing a chassis where all of the parts, including the four motors and the batteries, slot into.

Skills

  • Remote control
  • Using a motor driver
  • Practical 3D printing for use in a larger sytem
