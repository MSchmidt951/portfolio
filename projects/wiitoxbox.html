Wii Controller on Xbox

Description

This project allows a wii remote to be used on an Xbox.
It does this by connecting a wii remote to a laptop via bluetooth, then the laptop to the Xbox.

Using a wii remote rather than a normal controller allows for the use of the gyroscope/accelerometer measuring the angle at which the remote is held at.
This provides unique, fun gameplay and can feel more intuitive, for example while driving a car or flying a plane in game.
